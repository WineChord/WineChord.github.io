CST2018 1-7 FIFO

发表于 2019-02-09 | 分类于 THUOJ |

题目描述

  • 简化版的FIFO调度算法
    1. 访问某个地址的值时，先在缓冲器中寻找该地址及其对应值
    2. 若在缓冲器中找到该地址，则将其对应值返回，操作结束
    3. 若未找到，则从内存中读取该地址对应值，并试图将该地址及其对应值存到缓冲器中
    4. 若缓冲器未满，则存，操作结束
    5. 若满，删除最先进入缓冲器的元素，添加当前访问地址及其值
  • 刚开始缓冲器为空，现有一系列访存请求，问有几次需要去内存中访存

输入输出

  • 输入：
    • 第 $1$ 行：$n\ m$ 缓冲器的大小和访存次数
    • 以下一行：$m$ 个整数，表示每次访问的地址，每个地址是1-100的整数
  • 输出：
    • 一个整数，表示答案

数据范围及限制

  • $1\le n\le 100$
  • $1\le m\le 1000$

  • $1\le 内存地址\le 100$

  • 时间：1 sec

  • 空间：256 MB

提示

  • 队列

CST2018 1-5 Zuma

发表于 2019-02-09 | 分类于 THUOJ |

题目描述

  • 一条轨道上初始有 $n$ 个彩色珠子，其中任意三个相邻的珠子不会完全同色。
  • 可以发射珠子到轨道上，一旦有三个或更多同色的珠子相邻，就会立即消失。可发生连锁反应
  • 请计算一系列操作之后的珠子序列

输入输出

  • 输入：
    • 第 $1$ 行：由 ‘A’-‘Z’ 组成的字符串，表示初始珠子序列
    • 第 $2$ 行：$m$ 表示共有 $m$ 次操作
    • 以下 $m$ 行：每行代表一次操作，每次操作为 $k\ \Sigma$ ，其中 $k\in[0,b]$ 表示新珠子嵌入之后在轨道上的位序，$\Sigma$ 表示新珠子的颜色
  • 输出：
    • 共 $1$ 行：给出所有操作之后轨道上的珠子序列
    • 若没有珠子，则输出空行（一个换行符）

数据范围及限制

  • $0\le n\le 500,000$
  • $0\le m\le 500,000$

  • 输入保证位置合法

  • 时间：2 sec

  • 空间：256 MB

提示

  • 本题字母、数字混合输入，且初始序列可能为空，请使用足够鲁棒的读入方式。

  • [ 我们将介绍一种简单高效的数据结构，分块数组。他是一个结合了链表和列表的高效的数据结构。 ]

  • [ 列表可以 $O(1)$ 完成寻址，但是需要 $O(N)$ 完成删除和插入，链表可以 $O(1)$ 完成删除和插入，但是需要 $O(N)$ 完成寻址。我们将他们结合起来选择 $\sqrt{n}$ 为一个中间值。 ]

  • [ 将 $N​$ 长度的列表以 $O(\sqrt{n})​$ 长度断成 $O(\sqrt{n})​$ 块，每块用链表连起来，这样链表的长度为 $O(\sqrt{n})​$ 同时列表的长度为 $O(\sqrt{n})​$，我们可以在 $O(\sqrt{n})​$ 的时间范围内完成寻址插入和删除。 ]

  • [ 另一个需要注意的是在移动数组元素时，使用 memove 能够大大提升程序的性能 ]

Ch11 Paradigms

11.1 Supervised Learning Paradigm

  • 接受一组标注的样本作为训练数据，然后对未知点进行推测
  • 6个步骤：
    • 1）确定训练类型
    • 2）收集训练集
    • 3）确定特征提取方法
    • 4）设计该任务的算法
    • 5）训练该算法
    • 6）评估精确性
  • 相关任务：
    • 分类，回归，排名

11.2 Unsupervised Learning Paradigm

  • 接受未标注数据，然后对未知点做出预测
  • 目标是发现数据中共性的东西
  • 相关任务：
    • 聚类/密度估计/降维

11.3 Reinforcement Learning Paradigm

  • 学习器是一个决策制定智能体，在环境下采取行动并获得这些动作的回报
  • 经过一系列试错运行之后，可以学到最优策略
  • 目标是使回报最大化
  • 不提供输入和输出对，专注于在线的性能优化
  • 类型
    • 1）基于模型
      • 首先以马尔可夫决策过程来行动，并学习 T 和 R
      • 然后用学习的 T 和 R 进行数值迭代或策略迭代
    • 2）无模型
      • 学习策略没有任何模型
      • 避开学习 T 和 R 的过程，采用直接评估策略
      • 基于预测的时间差分（temporal difference）法
  • 1）基于模型的强化学习
    • 思想：
      • 通过实践经验学习模型。若学到的模型正确，则用于数值求解
  • 2）无模型强化学习
    • Actor-Critic methods 动作者·评判者方法
      • 策略迭代（On-policy）的时间差分版
      • 一种明确表示独立于价值函数的策略的结构
      • 策略（动作者）用于选择动作
      • 价值函数（评判者）用于评估动作者所完成的动作
        • TD error: $\delta_t=r_{t+1}+\gamma V(S_{t+1})-V(S_t)$
        • Preference: $p(s_t,a_t)\leftarrow p(s_t,a_t)+\beta\delta_t$
    • Q-learning
      • 价值迭代（Off-policy）的时间差分版
      • 基于回报值和 Q-value 函数，递增估计动作的 Q 值
    • 更新规则是一种时间差分学习的变体，采用 Q 值与内置的下个状态 Q 值得最大运算符：
      • $Q(s_t,a_t)=\sum_aT(s_t,a_t,s_{t+1})[R(s_t,a_t,s_{t+1})+\gamma\max_{a_{t+1}}Q(s_{t+1},a_{t+1})]$
    • 学习到基于样本的，东顾总-值函数 Q
  • 新算法
    • Deep Q-Network
    • Deterministic Policy Gradients (DPG)
    • Asynchronous Advantage Actor-Critic (A3C)
    • UNsupervised REinforcement and Auxiliary Learning
    • Neural Episodic Control (NEC) 神经情景控制
  • 应用
    • 机器人
      • 手臂
      • 导航
    • 游戏

11.4 Other Learning Paradigms

  • Ensemble learning 集成学习
    • 多个弱学习器组成一个强学习器
  • Learning to learn 学会学习
    • 基于先前的经验学习归纳偏差
  • Transfer learning 迁移学习
    • 将已有的知识用于不同但相关的问题
  • Adversarial learning 对抗式学习
    • 以一种对抗性方式（即零和博弈）来生成模仿某种分布的数据
  • Collaborative learning 协同式学习
    • 以某种协同式（如非零和博弈，双赢）来得到所期待的输出结果

12.1 Probabilistic Models

  • 概率论研究不确定性问题

  • 概率模型是采用概率论来表示所有不确定的形式

  • 贝叶斯规则允许我们推断未知量，适配我们的模型，做出预测并从数据中学习

  • 典型方法

    • 朴素贝叶斯和贝叶斯网络
    • 概率规划
    • 高斯过程
    • 隐马尔可夫模型

  • Often the goal of machine learning is to determine the probability of an event, e.g. $P(a\ person\ is\ at\ risk\ for\ a\ disease)$

  • 两个主要学派

    • 频率学派
    • 贝叶斯学派
      • 给定一组特定数据，概率对事件的不确定性进行量化，旨在最接近事实真值

  • Bayes Rule

    • 告诉我们如何从数据推断假设

    • 学习和预测可以看作是推理的形式

    • 推断未知的量 => 适配我们的模型 => 做出预测 => 从数据中学习

    • $$
      P(hypothesis|data)=\frac{P(data|hypothesis)P(hypothesis)}{P(data)}
      $$

  • Bayesian ML

    • 任何事物都遵循以下两规则

      • 求和规则：$P(x)=\sum_yP(x,y)$
      • 乘积规则：$P(x,y)=P(x)P(y|x)$

    • $P(\theta|\lambda,m)$: posterior probability of given data $\lambda$ 后验概率

    • $P(\lambda|\theta,m)$: likelihood of parameters $\theta$ in model m

    • $P(\theta|m)$: prior probability of $\theta$ 先验

    • $P(\lambda|m)$: evidence

    • $$
      P(\theta|\lambda,m)=\frac{P(\lambda|\theta,m)P(\theta|m)}{P(\lambda|m)}
      $$

    • $$
      posterior=\frac{likelihood\times prior}{evidence}
      $$

  • 参数方法/非参数方法

    • 参数方法
      • 设有限参数集 $\theta$ ，给定该参数，则特征预测 $x$ 独立于观测数据 $D$，即
        • $P(x|\theta,\lambda)=P(x|\theta)$
      • 因此，捕捉任何事物在于要知晓这些数据
      • 故即使数据量无限，模型复杂度也是有界的
      • 故参数方法不是很灵活
    • 非参数方法
      • 设数据分布不能用这样一个有限参数集来定义，但通常能够由假设的无限维度 $\theta$ 来定义。通常我们把 $\theta$ 看作是一个函数
      • $\theta$ 可以捕捉数据 $D$ 的信息量可以随着数据量的增长而增长
      • 故而更加灵活

12.2 Geometric Models

  • 直接在实例空间构建，空间可以被认为是几何的概念，如：
    • 欧氏几何：线，面，欧式距离
    • 黎曼几何：流形，黎曼距离
  • 二维/三维的模型易于可视化
  • 线和面
    • 使用线或平面这样的几何模型来构建一些机器学习算法
  • 流形
    • 一种拓扑空间，其每个点附近存在近似的欧几里得空间
    • 更精确地说，一个 $n$ 微流形的每个点都是一个临近点，它与维度为 $n$ 的欧几里得空间同胚。
    • 高维输入数据中可以发现其固有的低维特性
    • 低维流形假设
      • 高维输入数据依附于一个低维流形
    • 典型的流形学习算法
      • Isomap (Isometric mapping 等距映射)
      • LLE (Locally Linear Embeddings 局部线性嵌入)
      • LE (Laplacian Eigenmaps 拉普拉斯特征映射)
      • LTSA (Local Tangent Space Alignment 局部切空间对齐)
      • Inductive Manifold Learning 归纳流形学习
  • Isomap (Isometric mapping) 等距映射
    • 保留数据之间的测地线距离而不是欧几里得距离
    • 算法流程
      • 确定每个点的相邻点
      • 构建一个邻接图
      • 计算两点之间的最短距离
      • 计算低维嵌入，采用多元尺度分析（MDS）方法
  • LLE (Locally Linear Embedding) 局部线性嵌入
    • 通过在小邻域中找到最佳线性重构，将数据点嵌入在低维空间中
    • 计算每个点的重建权重，然后通过求解特征值问题对嵌入成本进行最小化
  • LE (Laplacian Eigenmaps) 拉普拉斯特征映射
    • Le 重申在非线性映射问题作为一种图的顶点嵌入，并且采用图拉普拉斯来生成一个平滑的映射。
  • 典型的流形学习应用
    • 图像处理
    • 基因表达图谱
    • 手写体识别
    • 人脸识别

12.3 Logical Models

  • 易于解释的逻辑表达式，易于转换成人类能够理解的规则
  • 典型的逻辑模型
    • 一阶逻辑
    • 关联规则
    • 决策树
  • 关联规则
    • 是一种用于在大型数据库中变量之间发现有趣关系的方法
    • 采用差异趣味性度量方式，旨在识别数据库中发现的强规则
    • 可用于发现超市产品之间的规律，称之为购物篮分析
      • ${onions,potatoes}\Rightarrow {burger}$
    • Def.
      • $I={i_1,i_2,…,i_n}$: a set of n attributes called items
      • $T={t_1,t_2,…,t_m}$: a set of transactions called transactions database.
      • Each $t_i$ is a subset of the items in $I$, i.e.,
        • $t_i\sube I$
      • $X\Rightarrow Y$: an association rule, where $X,Y\sube I$, and $X\cap Y=\varnothing$
  • 决策树
    • 使用决策树作为预测模型，将项的观察结果映射到关于项的目标值
    • 目标变量可以取一组有限值得树模型称为分类树；其中叶节点表示类标签，分支表示通往这些类标签的特征连接。

12.4 Networked Models

  • 人工神经网络 (ANN)
  • 是人脑的一种人工表征，试图模拟人类的学习过程
  • 通过互联的“神经元”构建一个系统，神经元之间相互发送消息
  • 神经元之间的连接具有数值权重，可以通过经验调整，使 ANN 适应输入并且能够学习。
  • 前馈神经网络
    • 信息从输入节点仅仅以一个方向即前进方向穿过隐藏层并抵达输出结点
  • 循环神经网络
    • 连接形成有向循环
    • 建立网络的内部状态，使之展现动态的时间特性
  • DNN
    • 使用许多层非线性处理单元，用于特征提取和转换
    • 能够学习数据的多层特征或表征。高层特征来自于底层特征
    • 成为更广泛的机器学习领域的一部分：学习数据表征
    • 学习多层级表征，对应于不同的抽象层级；这种层级形成了一种概念的层次结构
    • 代表性的DNN
      • Deep belief networks (DBN) 深度信念网络
      • Convolutional neural networks (CNN) 卷积神经网络
      • Deep Boltzmann machines (DBM) 深度波兹曼机
      • Recurrent neural networks (RNN) 循环网络
      • Long short-term memory (LSTM) 长短期记忆
      • Auto-Encoders 自动编码器
      • Generative Adversarial Network (GAN) 生成对抗网络
  • CNN
    • 前馈式ANN，使用至少一个卷积层来代替一般的矩阵乘法
    • 四个关键思想
      • local connections (convolution) 局部连接（卷积）
      • shared weights 共享权值
      • pooling (sampling) 池化（采样）
      • many layers
    • conv layer
      • 包含一组学习滤波器，每个滤波器对输入的宽和高进行卷积，计算滤波器和输入之间的点积，生成一个该滤波器的二维活动图
    • pooling layer
      • 是一种非线性下采样，将输入图像分割成一组不重叠的矩形，对每个这样的子区域，再产出其最大值。
  • GAN
    • 2014 Ian Goodfellow
    • Adversarial Network, inspired by Adversarial game
    • Generator maps from a latent space to a particular data distribution of interest
    • Discriminator discriminate between real samples and samples produced by generator
