169. Majority Element-12

发表于 2018-11-02 | 分类于 LeetCode |

问题描述

寻找众数（出现次数大于$\lfloor n/2 \rfloor$ ）

方法一：暴力

对每一个元素都数一下它自己出现的次数

time: $O(n^2)$, space: $O(1)$

方法二：HashMap

使用 HashMap 来数每个元素出现的个数

time: $O(n)$

space: $O(n)$

方法三：排序

将数组进行排序，那么众数应该出现在 $\lfloor n/2 \rfloor$（n是奇数），或者 $\lfloor n/2 \rfloor+1$ （n为偶数）

方法四：随机

随机选取一个数，判断它是不是众数

方法五：分治法

方法六：Boyer-Moore Voting

从前向后遍历数组，首先把当前的元素认为是众数，用一个变量 count 来表示这个认为是众数的数出现的次数。往后依次处理元素，假如碰到了这个数，count加一，假如碰到的不是这个数，count减一，当 count 减为 0 时，重新选取下一个元素作为众数。相当于不断舍弃一个前缀，这个前缀中可能的众数和非众数的数量是相同的，所以可以保证的是即使舍弃的前缀中包含众数，这个众数在后缀中仍为众数。

方法七：Bit manipulation

找出这n个数，32bit中每个bit最多的的那一种情况

448. Find All Numbers Disappeared in an Array-11

发表于 2018-11-02 | 分类于 LeetCode |

问题描述

给一个数组，找到数组中按顺序缺少的元素。（数组长度为n，数组中元素的取值范围为[1,n]（说明有重复元素））

实现

遍历数组用数组中的元素值作为下标，将该下标对应的数组元素的绝对值取负。这样到最后所有访问过的下标中，其对应的数组元素值为负，那些元素值为正的说明其下标未被访问过，因此该下标对应的元素不存在。

碰到一个非0时只需与前面的进行交换。
