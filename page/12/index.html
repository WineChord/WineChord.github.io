136. Single Number-HT-7

发表于 2018-11-03 | 分类于 LeetCode |

问题描述

给一个非空的整数数组，除了一个元素出现了一次之外每个元素都出现了两次。找出那个只出现一次的元素。

方法一：List operation

  • time: $O(n^2)$
  • space: $O(n)$

方法二：Hash Table

  • time: $O(n)$
  • space: $O(n)$

方法三：数学

$$ 2 \times (a+b+c) - (a+a+b+b+c) = c $$

  • time: $O(n+n)=O(n)$, see it as sum(list(i for i in nums)
  • space: $O(n+n)=O(n)$

方法四：位操作

  • time: $O(n)$
  • space: $O(1)$
