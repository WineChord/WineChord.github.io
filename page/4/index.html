703. Kth Largest Element in a Stream-Heap-1

发表于 2018-11-04 | 分类于 LeetCode |

问题描述

设计一个类来找到一个流中第 k 大的元素。类初始的时候有 k 和一个数组 nums。之后每进行添加元素的操作时都要能够返回第 k 大的元素。

解法

思路：建立一个只含有 k 的比较大元素的最小堆。
对于 add:

  • 比较新加入的元素 x 和堆中的最小值，决定我们是否要 pop min insert x
  • 并且要考虑 heap_size 小于 k 的情况
    对于构造，只需调用 N 次 add 即可
