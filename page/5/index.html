136. Single Number-HT-7

发表于 2018-11-03 | 分类于 LeetCode

问题描述

给一个非空的整数数组，除了一个元素出现了一次之外每个元素都出现了两次。找出那个只出现一次的元素。

方法一：List operation

  • time: $O(n^2)$

  • space: $O(n)$

    方法二：Hash Table

  • time: $O(n)$

  • space: $O(n)$

    方法三：数学

    $$
    2 \times (a+b+c) - (a+a+b+b+c) = c
    $$

  • time: $O(n+n)=O(n)$, see it as sum(list(i for i in nums)

  • space: $O(n+n)=O(n)$

    方法四：位操作

  • time: $O(n)$

  • space: $O(1)$

575. Distribute Candies-HT-6

发表于 2018-11-03 | 分类于 LeetCode

问题描述

给一个长度为偶数的整数数组，不同的数字代表不同类型的糖果。将糖果均分给哥哥姐姐，问姐姐可以得到的糖果种类总数最大是多少？

方法一：暴力

生成糖果的所有排列，然后确定前半个数组中的种类个数。

  • time: $O(n!)$

  • space: $O(n)$

    方法二：更好的暴力

    观察到姐姐最多可以拿到的种类数是 n/2，对于种类数少于 n/2 的情况，我们可以把所有种类数的糖果都给姐姐。所以我们遍历数组，找出糖果的种类总数。

  • time: $O(n^2)$

  • space: $O(1)$

    方法三：使用排序

  • time: $O(nlogn)$

  • space: $O(1)$

    方法四：使用 set

  • time: $O(n)$

  • space: $O(n)$
