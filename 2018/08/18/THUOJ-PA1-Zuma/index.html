THUOJ PA1 Zuma

发表于 2018-08-18 | 分类于 THUOJ |

Description

Zuma is a game that has a sequence of beads. All the beads are colored but no three adjacent ones are allowed to be with a same color. Now we insert new bead into the sequence. Once three or more beads with a same color become adjacent due to an insertion, they will vanish immediately.

Input

  • First line gives the initial bead sequence(letter from ‘A’ to ‘Z’),
  • Second line gives the number of insertions n
  • The following n lines tell all the insertions in turn. Each contains an integer k and a capital letter.

Output

The evolutionary history of the bead sequence.
Specially, “-“ stands for an empty sequence.

Restrictions

  • 0 <= n <= 10^4, 0 <= length of the initial sequence <= 10^4
  • Time: 2s
  • Memory: 256 MB

Idea

Use list. First construct a linknode and then form a list class. Use check() function to check whether there are three or more adjacent beads. While checking is true, call change() function to clean the adjacent beads.
The key algorithm is the implementation of list.
