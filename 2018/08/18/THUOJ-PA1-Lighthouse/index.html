THUOJ PA1 Lighthouse

Posted on 2018-08-18 | In THUOJ

Description

If another lighthouse is in grey area, they can beacon each other.

Input

  • First line: the total number of lighthouse N
  • Second to (N+1)th line: lighthouse is on the point (X, Y)

Output

How many pairs of lighthouses can beacon each other.

Restrictions

  • 90% : 1 <= n <= 3 * 10^5
  • 95% : 1 <= n <= 10^6
  • All : 1 <= n <= 4 * 10^6
  • For every lighthouses, X coordinates won’t be the same, Y coordinates won’t be the same.
  • 1 <= x, y <= 10^8
  • Time: 2s
  • Memory: 256 MB

Example

  • input

  • output

Idea

It is trivial to think that use two for loops can solve the problem. However the time consumption will be too large. First construct a point struct. Use quick sort to make the point is sorted according to it X coordinate. Then calculate the inversions in the Y coordinates. The total inversions mean that they cannot form lighthouses that beacon each other. Thus the result equals to the total permutation number minus the total inversions.
The hard point is how to calculate inversions. Here uses a variation of merge sort. When doing mergesort to their Y coordinates, keep a count value to record inversions.
