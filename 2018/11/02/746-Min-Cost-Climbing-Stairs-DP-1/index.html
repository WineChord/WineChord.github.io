746. Min Cost Climbing Stairs-DP-1

发表于 2018-11-02 | 分类于 LeetCode

问题描述

走楼梯，给一个数组，每个值是对应台阶所需要的花费，每次可以走一级或两级，问怎么走才能使花费最少？

使用动态规划

递推式：
$f[i]=cost[i]+min(f[i+1],f[i+2])$。
f[i]为从第i级台阶往后走的最大花费。
倒着推：

time: O(N), space: O(1)

recursive top-down with memo
