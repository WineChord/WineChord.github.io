Ch9 Perspectives about Machine Learning

9.1 What is Machine Learning

  • 是人工智能的一个分支，从事构建和研究可以从数据中学习的系统

  • Artificial Intelligence

    • 研究感知外部环境并为某个目标采取行动的“智能体”

  • Machine Learning

    • 人工智能的一个分支，从事构建可以从数据中学习的系统

  • 与其他学科的关系

    • 统计学习
      • 取自统计学的机器学习框架
    • 模式识别
      • 识别数据中的模式（$\approx$ 机器学习+数据模式）
    • 数据挖掘
      • 发现数据中的未知特性（$\approx$ 机器学习+数据库）
    • 计算机视觉
      • 从图像中提取信息（$\approx$ 机器学习+图像处理）

  • 两种通用的学习类型

    • Inductive Learning 归纳学习（bottom-up logic）
      • 从特定的训练实例中获得或发现通用的规则或事实
    • Deductive Learning 演绎学习（top-down logic）
      • 使用一套已知的规则和事实去推导适合该训练数据的猜测

  • 机器学习的形式化定义

    • Tom Mitchell

      • 1997:

      A computer program is said to learn from experience E with respect to some class of tasks T and performance measure P, if its performance at tasks in T, as measured by P, improves with experience E.

9.2 History of Machine Learning

  • 三个学派
    • Connectionism 联结主义
      • Perceptron, ANN, DL
    • Symbolicism 符号主义
      • Association Rules, Decision tree, Random Forests
    • Behaviorism 行为主义
      • Reinforcement learning

9.3 Why Different Perspectives

  • 难点
    • 应该选择哪种算法
    • 有多少种算法
    • 难点是什么
    • 一个视角够吗?
  • 三个观点
    • 做什么 $\Rightarrow$ 问题 $\Rightarrow$ 学习任务
      • 任务：表示可以用机器学习解决的基本问题
    • 什么形式 $\Rightarrow$ 场景 $\Rightarrow$ 学习范式
      • 范式：表示机器学习中发生的典型场景
    • 如何做 $\Rightarrow$ 手段 $\Rightarrow$ 学习模型
      • 模型：表示可以处理完成一个学习任务的方法

9.4 Three Perspectives on Machine Learning

9.4.1 Learning Tasks

  • 学习任务用于表示可以用机器学习解决的基本问题
Tasks Brief Statements Typical algorithm
Classification 分类 将输入分成多个类别 SVM 支持向量机
Regression 回归 输出是连续的而不是离散的 Bayesian linear regression 贝叶斯线性回归
Clustering 聚类 输入被划分为若干事先位置的组 k-means
Ranking 排名 将值来换成排名 PageRank
Density estimation 密度估计 寻找某个空间中输入的分布 Boosting Density Estimation 增强式密度估计
Dimensionality reduction 降维 通过将输入映射到低维空间来将其简化 Isomap 等距特征映射
Optimization 优化 从所有可能的解中寻找最优解 Q-learning

9.4.2 Learning Paradigms

  • 用于表示机器学习中发生的典型场景
  • 根据机器学习的典型场景或样式来区分学习范式
    • 它怎样从数据中学习
    • 它如何同环境互动
Paradigms Brief Statements Typical Algorithm
Supervised 有监督 采用一组标注好的数据进行训练，再对未知点进行预测 SVM
Unsupervised 无监督 仅接受未标注数据，再对所有的未知点做出预测 k-means
Reinforcement 强化 算法与外部环境交互，每个动作得到一个回报 Q-learning

9.4.3 Learning Models

  • 用于表示可以处理完成一个学习任务的方法
  • ML的效果在很大程度上取决于解决该学习任务时所选用的方法
Models Brief Statements Sub-models Typical Algorithm
Geometric 几何 采用线，面，距离或流形来构建学习算法 线，面，距离，流形(Manifold) 线性回归，SVM，k-NN，Isomap等距映射
Logical 逻辑 采用逻辑模型 逻辑，规则 Inductive Logic Program 归纳逻辑编程，Association Rule 相关规则
Networked 网络 采用网络模式 浅层，深层 Perceptron 感知机，CNN 卷积神经网络
Probabilistic 概率 采用概率模式来表示随机变量之间的条件相关性 Bayes贝叶斯，Generative 生成，Statistic 统计 贝叶斯网络，概率规划，线性回归
