Ch10 Tasks in Machine Learning

10.1 Classification

  • 是基于包含已知类别成员观测值的训练数据集，来辨识新的观测值属于哪一组类别的任务

10.1.1 How Classification Works

  • Classifier

    • 一种实现分类，尤其是构成一种具体实现的算法，被称为一个分类器
    • 有时还指的是由分类算法所实现的数学函数，它将输入数据映射为一个类别

  • A Formal Description of Classification

    • Let $\mathbb{R^n}(n\ge1)$ denote a set of n-dimensional real-valued vectors, input space $\mathcal{X}$ is a subset of $\mathbb{R^n}$ , output space $\mathcal{Y}$ is a set of categories, $\mathcal{D}$ is an unknown distribution over $\mathcal{X}\times \mathcal{Y}$, then:

      • Let target labeling function: $f:\mathcal{X}\rightarrow \mathcal{Y}$

      • Training set (Labeled training sample set):

        • $$
          \mathcal{S}={(x^{(i)},y^{(j)})|(x,y)\in \mathcal{X}\times \mathcal{Y},i\in [1,m], j\in [1,n]}
          $$

      • Classification algorithm

        • Let a hypothesis set $\mathcal{H}$ are the mapping $\mathcal{X}$ to $\mathcal{Y}$, to determine a hypothesis (classifier function):
        • $h:\mathcal{X}\rightarrow\mathcal{Y}\ and\ h\in\mathcal{H}\ with\ small\ generalization\ error:\ R(h)=Pr_x[h(x)\ne f(x)]$

    • Classification:

      • Given a testing dataset of unknown categories:
        • $\mathcal{X}={x^{(i)}|x\in\mathcal{X},i\in[1,m]}$
      • Using the classifier function $h(\mathcal{X})=\mathcal{Y}$ determined at above to predicate classifying results:
        • $\mathcal{y}=h(\mathcal{X})={y^{(j)}|y\in\mathcal{Y},j\in[1,n],h(x)=y}$
        • where $\mathcal{Y}$ is the set of known categories

10.1.2 Linear and Nonlinear

Linear Classification
  • 是通过线性分类器来进行分类
  • 是一个具有线性决策边界的线性判别函数
    • $H={\mathbb{x}\xrightarrow\ y(\mathbb{x})=\mathbb{w}\cdot \mathbb{x}+b|\mathbb{w}\in \mathbb{R^n},b\in \mathbb{R}}$
Non-linear Classification
  • 是通过一个非线性分类器来进行分类
  • 若干非线性决定边界，并且可能是非连续决定边界
    • 如在SVM中的非线性分类器是一个非线性核函数

10.1.3 Dimensions and Classes

  • Dimensions 维度
    • 若问题空间的维度为 $n$，则其线性分类器的维度为 $n-1$ 的超平面
  • Classes 类别
    • Two classes: k=2
    • Multiple classes: k>2
  • Case Study: Softmax Classifier
    • It is a multiclass classifier, implemented by a softmax function
    • Softmax function maps a K-dimensional vector $\mathbb{x}$ of arbitrary real values to a K-dimensional vector $\sigma(\mathbb{x})$ of real values (range 0 to 1, add up to 1).
      • $\sigma(\mathbb{x})_j=\frac{e^{x_j}}{\sum_{k-1}^Ke^{x_k}}\ j=1,…,K$
    • In probability theory, the output of the softmax function can be represented as a categorical distribution, i.e., a probability distribution over K different outcomes.
      • $P(y=j\ |\mathbb{x})=\frac{e^{\mathbb{x}^T\mathbb{w}j}}{\sum{k-1}^Ke^{\mathbb{x}^T\mathbb{w}_k}}$
    • 已用于多种多元分类方法
      • multinomial logistic regression 多项式逻辑回归
      • multiclass linear discriminant analysis 多元线性判别分析
      • naive Bayes classifiers 朴素贝叶斯分类器
      • ANN
      • RL

10.2 Regression

  • 回归分析是估计变量间关系的统计过程，它包含对多变量进行建模和分析的许多技术，焦点是某个自变量与一个或多个因变量之间的关系
  • 要解决输出是真实连续值得问题
  • 预测每个项的真实值
  • 回归 vs. 分类
    • 相似点：都需要训练过程
    • 不同点：回归连续，分类离散

10.2.1 How Regression Works

  • Let $\mathbb{R}^n(n\ge1)$ denote a set of n-dimensional real-valued vectors, $\mathbb{R}+$ is a set of non-negative real numbers, input space $\mathcal{X}$ is a subset of $\mathbb{R}^n$, output space $\mathcal{Y}$ is a set of real numbers $\mathbb{R}+$, $D$ is an unknown distribution over $\mathcal{X}\times \mathcal{Y}$ , then
    • Let target labeling function:
      • $f:\mathcal{X}\rightarrow \mathcal{Y}$
    • Training set(Labeled training sample set):
      • $\mathcal{S}={(x^{(i)},y^{(i)})|(x,y)\in \mathcal{X}\times\mathcal{Y},i\in[1,m]}$
    • Regression algorithm:
      • Given hypothesis set $H$, to determine a hypothesis (regression function):
      • $h:\mathcal{X}\rightarrow \mathcal{Y}\ and\ h\in H$
      • with small generalization error $R(h)$:
        • $R(h)=E_x[L(h(x),f(x))]$
    • Regression
      • Given a testing dataset of unknown output:
        • $\mathcal{X}={x^{(i)}|x\in\mathcal{X},i\in[1,m]}$
      • Using the regressive hypothesis $h(\mathcal{X})=\mathcal{Y}$ determined at above to predicate regressive results:
        • $\mathcal{R}=h(\mathcal{X})={y^{(i)}|y\in \mathcal{Y},i\in [1,n],h(x)=y}$
      • Note, in which:
        • $\mathcal{Y}$ is a set of real continues numbers

10.2.2 Linear and Nonlinear

  • Linear Regression
    • linear combination of the model parameters
  • Nonlinear Regression
    • nonlinear combination of the parameters

10.3 Clustering

10.3.1 How Clustering Works

  • 将数据对象进行分组
  • 聚类最大的不同点在于他是一种无监督学习，没有训练数据
  • 基于距离，密度等发现聚类
  • A Formal Description of Clustering
    • Let $\mathbb{R}^n(n\ge1)$ denote a set of n-dimensional real-valued vectors, input space $\mathcal{X}$ is a subset of $\mathbb{R}^n$, output space $\mathcal{Y}$ is a set of unknown clusters, $D$ is an unknown distribution over $\mathcal{X}\times \mathcal{Y}$, then:
    • Let a clustering function:
      • $h:\mathcal{X}\rightarrow\mathcal{Y}\ and\ h\in H$
    • Clustering:
      • Given a testing set of unknown clusters:
        • $\mathcal{X}={x^{(i)}|x\in \mathcal{Y},i\in [1,m]}$
      • Using the clustering function determined at above to analyze the clustering results:
        • $\mathcal{Y}=h(\mathcal{X})={y\in \mathcal{Y},i\in [1,n],h(x)=y}$

10.3.2 Major Approaches of Clustering

  • 1) Connectivity-based clustering 连接性
  • 2) Centroid-based clustering 中心点
  • 3) Distribution-based clustering 分布
  • 4) Density-based clustering 密度

10.5 Dimensionality Reduction

  • 将输入映射到低维空间
  • 降维的原因：
    • 在高维空间对数据进行分析时会发生维度灾难
    • 数据稀疏或无关
    • 二维或三维数据更容易表示
  • 线性降维/非线性降维
  • 方法：
    • Principal Component Analysis (PCA) 主成分分析
    • Linear Discriminate Analysis (LDA) 线性判别分析
    • Multilinear subspace learning 多线性子空间学习
      • Multilinear PCA (MPCA) 多线性主成分分析
      • Multilinear Linear Discriminant Analysis (MLDA) 多线性线性判别分析
  • PCA 举例
    • 是一种统计过程
    • 采用正交变换将一组可能相关变量的观测值转换成一组称为主成分的线性不相关变量
    • 主成分的数量小于原始变量的数量
  • 非线性降维的方法
    • Multi-dimensional Scaling 多元尺度分析
      • 经典/度量/非度量/广义
    • Kernel approaches 核方法
      • 核PCA
      • Kernel Fisher Discriminant Analysis (KFD) 核费希尔判别分析
    • Manifold learning approaches 流形学习方法
      • Isometric feature mapping (Isomap) 等距特征映射
      • Locally-linear embedding (LLE) 局部线性嵌入
  • Multi-dimensional Scaling (MDS) 多元尺度分析
    • 用于数据可视化的一些相关统计技术，用来考察数据中的相似性和非相似性
    • 构建一个所有点之间的成对距离矩阵，然后在低维空间计算每个点的位置，便于二维或三维可视化
