CST2018 1-1 A+B problem

发表于 2019-02-09 | 分类于 THUOJ |

题目描述

  • 实际上是 $A*B$ 问题，和PA2-1有关。
  • 问题是模拟 Python 中自带的长整型
  • 可以计算几百位的整数乘法

输入输出

  • 输入：
    • 第 $1$ 行：​$n$ # 要计算 ​$n$ 组乘法
    • 以下 $n$ 行：每行包含两个非负整数 $a$ $b$
  • 输出：
    • 共 $n$ 行：每行输出乘积

输入输出样例

  • 此样例是第一个测试点

数据范围及限制

  • $1\le n\le 500​$

  • $0\le a\lt 10^{5000}​$

  • $0\le b\lt10^{5000}​$

  • 时间限制：1 sec

  • 内存限制：256 MB

提示

  • 使用高精度算法

思路

  • 实现一个高精度乘法即可
